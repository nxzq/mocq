## Verbose Mode

:::tip

when debugging set the `MOCQ_VERBOSE` .env variable to `true` for verbose logs

:::

### node (server-side code)

import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

<Tabs>
  <TabItem value="env" label=".env file" default>
    ```js
    export MOCQ_VERBOSE=true // note: must `source .env` after setting value
    ```
  </TabItem>
  <TabItem value="code" label="inline code">
    ```ts
    process.env.MOCQ_VERBOSE = 'true'
    ```
  </TabItem>
  <TabItem value="cli" label="inline cli">
    ```bash
    MOCQ_VERBOSE=true # whatever cli command | example: MOCQ_VERBOSE=true bun test
    ```
  </TabItem>
</Tabs>
